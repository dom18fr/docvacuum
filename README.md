# Docvacuum

## Installation
### Prerequisites
- Vagrant
- Ansible
- VitualBox
- Composer
- (vagrant-hostsupdater)

### Instalation
1. Clone repository
2. `composer update` in `box` folder
3. Add custom configuration in `box/config/config.yml` (See `default.config.yml`)
4. Launch `vagrant up` in `box` folder
